## 🚀 Features

- 🔍 **Scan Project**
  - Quickly analyze your project and detect unused files.

- 🗑️ **Trash Support or Accidental Files Delete**
  - Delete orphaned assets and stores in Trash
  - Automatically Retrive from Trash

- 📊 **Detailed Overview**
  - Shows total orphaned files and their combined size.
  - Groups assets by type (e.g., images, unknown, etc.).

- 📂 **Interactive File List**
  - View all orphaned files in a clean, scrollable panel.
  - Displays file paths and sizes.
  - Select files for inspection.

- 🔎 **Search Functionality**
  - Easily filter files using the search bar.

- 📤 **Sorts Results - Size, Name, Type of Files**
  - Sorts files results.

- 👁️ **Preview (Optional)**
  - Preview selected assets ( Image ) before taking action.
